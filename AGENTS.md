# Transcriber

Personal social media video transcription tool.

## Stack
- Flask (Python) backend
- yt-dlp for video/audio downloading
- faster-whisper for local AI transcription
- Single-file HTML/CSS/JS frontend

## Running
```bash
./run.sh
# or
source venv/bin/activate && python app.py
```

Opens at http://localhost:5050

## Supported Platforms
Instagram Reels, TikTok, YouTube/Shorts, Twitter/X, Facebook, LinkedIn, and any yt-dlp supported site.
